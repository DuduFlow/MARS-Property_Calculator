# 不動產案例稅賦分析

互動式不動產稅務試算工具，包含兩個計算機：

- **父母贈與房地產稅賦分析** — 比較 6 種贈與方式，依不同持有年限（2年內／2–5年／5–10年／超過10年），計算贈與稅、房地合一稅與土地增值稅的總稅負，並標示最省稅方案。
- **特定傳承與稅賦分析** — 將財產傳承給某特定子女，比較「生前贈與 vs 死後繼承」在單純傳承、考慮出售、傳承規劃三種情境下的總稅負與財產分配比例。

> 版本：114.11　·　齊名揚老師編著

## 功能特色

- 完全互動，輸入即時重算（前端 JavaScript，無需後端）
- 結論前置摘要、稅負熱度比較表、長條圖與圓餅圖視覺化
- 深色／淺色（報告模式）切換，預設淺色適合簡報與列印
- 情境分享網址（輸入值編碼進網址，可存書籤或傳給同事）
- 一鍵匯出 PDF，作為客戶講義
- 輸入區忠實還原原始試算表：白底＝可輸入、藍底＝自動計算、灰底＝法定固定值

## 線上瀏覽（GitHub Pages）

1. 在 GitHub 建立一個新的 repository，將本資料夾所有檔案上傳。
2. 進入 repo 的 **Settings → Pages**。
3. **Source** 選擇 `Deploy from a branch`，Branch 選 `main`、資料夾選 `/ (root)`，按 Save。
4. 等待約 1–2 分鐘，即可透過 `https://<你的帳號>.github.io/<repo 名稱>/` 開啟。

## 本機使用

直接用瀏覽器開啟 `index.html` 即可。圖表函式庫（Chart.js）以 CDN 載入，需要網路連線。

## 技術說明

- 單一 HTML 檔，內含所有 CSS 與 JavaScript，無建置步驟。
- 圖表：Chart.js（CDN）。
- 所有稅務公式皆由原始 Excel 試算表逐格轉寫，並以 5,000 組隨機輸入交叉驗證，與 Excel 計算結果完全一致。
